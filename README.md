# Лабиринт Сокровищ


"Лабиринт Сокровищ" — это текстовая приключенческая игра, где игрок исследует таинственный лабиринт в поисках сокровищ. Игроку предстоит решать загадки, собирать предметы, преодолевать ловушки и находить путь к сундуку с сокровищами.


## Установка
Для установки проекта выполните команду:
```bash
make install
```
или
```bash
poetry install
```
## Запуск игры

```bash
make project
```

или

```bash
poetry run project
```

## Как играть

### Доступные команды:

- `go <direction>` или `north/south/east/west` — перейти в указанном направлении
- `look` — осмотреть текущую комнату
- `take <item>` — поднять предмет
- `use <item>` — использовать предмет из инвентаря
- `inventory` — показать содержимое инвентаря
- `solve` — попытаться решить загадку в комнате
- `help` — показать список доступных команд
- `quit` — выйти из игры
